Markdown语法之复选框

Mon Feb 08, 2021 in  日常  Markdown

categories: [日常, 学习]

学习内容：Markdown复选框

在无序列表符号后面加上[]或者[x]代表选中或者未选中情况

- [x] ROW1  
- [ ] ROW2

或者

+ [x] ROW1  
+ [ ] ROW2

或者

* [x] ROW1  
* [ ] ROW2

效果如下：

  • ROW1
  • ROW2
